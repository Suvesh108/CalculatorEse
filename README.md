# CalculatorEse

CalculatorEse is a comprehensive online calculator platform offering a suite of precise and easy-to-use calculators for financial planning, health tracking, and mathematical problem-solving. It is designed to be fully responsive, fast, and 100% free.

## Features

- **17+ Free Calculators**: Covering various daily and professional needs.
- **Categorized Sections**:
  - **💰 Financial**: Mortgage, Loan, Interest, Compound Interest, Investment, and Payment calculators.
  - **❤️ Health**: BMI, Calorie, BMR, and Body Fat calculators.
  - **🔢 Math**: Scientific, Percentage, Fraction, Age, Date, GPA, and Grade calculators.
- **Fully Responsive**: Optimized for both desktop and mobile devices.
- **Fast and Lightweight**: Built with standard HTML, CSS, and Vanilla JS.
- **Modern UI**: An intuitive, app-like interface.

## Tech Stack
- Frontend: HTML5, CSS3, Vanilla JavaScript
- Global UI Header/Sidebar Scripting (Injected via `ui.js`)

## Directory Structure
- `/css`: Contains stylesheets (`style.css` and `mobile.css`).
- `/js`: Contains JavaScript for UI injection and functional logic (`ui.js`).
- `/pages`: Separated category folders containing individual HTML calculator applications `/financial`, `/health`, and `/math`.
- `index.html`: The main landing page.

## SEO Config
- `sitemap.xml` and `robots.txt` are configured out-of-the-box for proper search engine indexing.
